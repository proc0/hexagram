# Hexagram

A hex tile merge mini game in Raylib for Raylib game jam

**build** and **run** using build script

#### Linux/Mac

```bash
chmod +x ./build.sh
./build.sh --run
```

#### Windows

```powershell
.\build.ps1 --run
```

## Requirements

- [CMake](https://cmake.org)
- [Emscripten](https://emscripten.org)
